## Usage:
--------------

``` bash
sudo python3 rpi_router.py <CLIENT_LINK_T> <CLIENT_ADDR> <GATEWAY_LINK_T> <GATEWAY_ADDR> 
```

- `CLIENT_ADDR`: Static remote network address;
- `GATEWAY_ADDR`: Static local gateway address (Raspberry Pi);
